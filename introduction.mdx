## Key Features

<AccordionGroup>
  <Accordion title="Creative Components" icon="sparkles">

    - **Data Reveal** - Animated sensitive content display with auto-hide
    - **Draggable Canvas** - Physics-based draggable interface with momentum
    - **Text Shift** - Smooth text transitions with staggered animations
    - **Creative Buttons** - Collection of animated button variants
  </Accordion>

  <Accordion title="Form Elements" icon="input-text">

    - **Smart Form** - Zod-powered forms with TanStack Query integration
    - **Multi-Step Form** - Guided step-by-step form interfaces
    - **File Uploader** - Drag-and-drop with previews and validation
    - **Password Strength Meter** - Visual password evaluation
  </Accordion>

  <Accordion title="Data Display" icon="table">

    - **Infinite Scroll** - Efficient virtual rendering for large datasets

    - **Notification Center** - Comprehensive notification management

## Built on Modern Standards

<Note>
  RigidUI components are built with accessibility, performance, and developer experience in mind. All components follow WAI-ARIA guidelines and are keyboard-navigable.
</Note>

RigidUI is powered by industry-standard technologies:

- **React & Next.js** - Modern React patterns with server and client components
- **TypeScript** - Full type safety throughout your application
- **Tailwind CSS** - Utility-first styling with complete customization
- **shadcn/ui** - Built on the popular component methodology
- **Zod** - Runtime type validation for forms and data

## Get Started in Minutes

RigidUI follows the shadcn/ui installation model - components are added directly to your project, giving you full control:

```bash
npx shadcn@latest add https://rigidui.com/r/smart-search.json
```

<Tip>
  Ready to build? Head over to the [Installation Guide](/installation) to set up RigidUI in your project.
</Tip>
